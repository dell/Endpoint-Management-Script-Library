This folder contains the following scripts. 
These scripts need to be run from the Intune Proactive Remediations but these scripts are only for detection and do not contain any remediation scripts.


1. Intune_Retreive_BIOS_version.ps1:
This script uses the DCIM_BIOSElement class to retrieve the BIOS version in json format.


2. Intune_Retreive_Manufacturer.ps1:
This script uses the DCIM_Chassis class to retrieve the Manufacturer info in json format.


3. Intune_Retreive_serviceTag.ps1:
This script uses the DCIM_Chassis class  to retrieve the service tag of the system in json format.
